# Edyfra

An advanced educational ecosystem built to deliver a seamless, high-performance learning experience. This repository contains the core application architecture, designed with a focus on modern full-stack development, clean UI patterns, and efficient state/data management.

---

## 🛠️ Tech Stack & Architecture

This project is built using a modern, robust, and scalable stack:

*   **Framework:** [Next.js](https://nextjs.org/) (App Router)
*   **Language:** [TypeScript](https://www.typescriptlang.org/) for strict type safety
*   **Database & ORM:** [Supabase](https://supabase.com/) (PostgreSQL) paired with [Prisma ORM](https://www.prisma.io/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) for fluid, utility-first layouts
*   **Font Optimization:** Using `next/font` with Geist Sans & Mono

### 1. Install Dependencies
```bash

# or
yarn install
# or

3. Run the Development Server
Bash

# or

# or

Open http://localhost:3000 in your browser to see the live local interface. You can begin exploring or modifying layout logic starting from app/page.tsx.
